# Dev Tools (Web UI only)

<div style="border-left: 6px dashed #cc0000; padding: 10px">
This functionality is not available via the VS Code Extension.
</div>
## Description

---
Tab is specifically designed to contain developer tools for the purposes of testing and debugging.
<div style="background-color: #fff1f0; border-left: 6px solid #ff4538; padding: 10px">
<b>Note:</b><br>
Tab is <b>only</b> available for non-production environments, and provided tools shall not serve to any other purposes besides testing and debugging.
</div>

## User Interface

---

### View "Dev Tools" Section.
By navigating to "Dev Tools", next tabs are going to be available:
- [MaaS](docs/04__Dev_Tools/1__MaaS/maas.md) - contains existing list of domains and their state.
- [Endpoint Mocks](docs/04__Dev_Tools/2__Endpoint_Mocks/endpoint_mocks.md) - allows to view and manage all existing mocking sets.
- [Diagnostic](docs/04__Dev_Tools/3__Diagnostic/diagnostic.md) - allows to execute system validations to find issues and get resolution hints.
